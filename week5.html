Les 5: GitHub & Netlify

Wat we hebben gedaan

In deze les leerden we hoe we onze website online konden zetten. We maakten een GitHub repository aan en koppelden deze aan Netlify, zodat de website automatisch gedeployed werd.

Belangrijke stappen

  • Een nieuwe GitHub repository aangemaakt.
  • Projectbestanden gecommit en gepusht.
  • Netlify gekoppeld aan GitHub.
  • Website automatisch online gezet.

Wat ik onthoud

  • Een repository is de online versie van je project.
  • Elke update op GitHub verschijnt automatisch op Netlify.
  • De juiste mappenstructuur is belangrijk.
